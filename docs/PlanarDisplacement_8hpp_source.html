60 /// \brief Two-dimensional Euclidean displacement vector in the XY plane. Contains two components in
61 /// Cartesian coordinates: x and y. Displacement is not to be confused with position; for a
62 /// two-dimensional Euclidean position vector in the XY plane, see PhQ::PlanarPosition. For a
63 /// three-dimensional Euclidean displacement vector, see PhQ::Displacement. For scalar displacement
64 /// components or for the magnitude of a displacement vector, see PhQ::Length.
