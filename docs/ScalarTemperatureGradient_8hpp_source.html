57 /// \brief Scalar temperature gradient component or magnitude of a temperature gradient vector. For
58 /// a three-dimensional Euclidean temperature gradient vector, see PhQ::TemperatureGradient. For a
59 /// two-dimensional Euclidean temperature gradient vector in the XY plane, see
60 /// PhQ::PlanarTemperatureGradient.
