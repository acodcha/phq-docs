45 /// \brief Two-dimensional Euclidean position vector in the XY plane. Contains two components in
46 /// Cartesian coordinates: x and y. Position is not to be confused with displacement; for a
47 /// two-dimensional Euclidean displacement vector in the XY plane, see PhQ::PlanarDisplacement. For
48 /// a three-dimensional Euclidean position vector, see PhQ::Position. For scalar position components
49 /// or for the magnitude of a position vector, see PhQ::Length.
