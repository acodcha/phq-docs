44 /// \brief Two-dimensional Euclidean temperature gradient vector in the XY plane. Contains two
45 /// components in Cartesian coordinates: x and y. For a three-dimensional Euclidean temperature
46 /// gradient vector, see PhQ::TemperatureGradient. For scalar temperature gradient components or for
47 /// the magnitude of a temperature gradient vector, see PhQ::ScalarTemperatureGradient.
