46 /// \brief Three-dimensional Euclidean position vector. Contains three components in Cartesian
47 /// coordinates: x, y, and z. Position is not to be confused with displacement; for a
48 /// three-dimensional Euclidean displacement vector, see PhQ::Displacement. For a two-dimensional
49 /// Euclidean position vector in the XY plane, see PhQ::PlanarPosition. For scalar position
50 /// components or for the magnitude of a position vector, see PhQ::Length.
