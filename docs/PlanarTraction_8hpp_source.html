49 /// \brief Two-dimensional Euclidean traction vector in the XY plane. Contains two components in
50 /// Cartesian coordinates: x and y. Traction is similar to pressure; however, traction can act in
51 /// any direction, whereas pressure always acts compressively perpendicular to a surface. For a
52 /// three-dimensional Euclidean traction vector, see PhQ::Traction. For scalar traction components
53 /// or for the magnitude of a traction vector, see PhQ::ScalarTraction.
54 template <typename Number = double>
