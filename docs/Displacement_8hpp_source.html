61 /// \brief Three-dimensional Euclidean displacement vector. Contains three components in Cartesian
62 /// coordinates: x, y, and z. Displacement is not to be confused with position; for a
63 /// three-dimensional Euclidean position vector, see PhQ::Position. For a two-dimensional Euclidean
64 /// displacement vector in the XY plane, see PhQ::PlanarDisplacement. For scalar displacement
65 /// components or for the magnitude of a displacement vector, see PhQ::Length.
