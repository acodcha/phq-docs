41 /// \brief Physical dimension set of a unit of measure or physical quantity. Composed of the seven
42 /// independent base physical dimensions: time (T), length (L), mass (M), electric current (I),
43 /// temperature (Î), amount of substance (N), and luminous intensity (J). Units of measure that
44 /// share the same physical dimension set are of the same type and can be converted between one
45 /// another. For example, the metre per second and the mile per hour are both units of measure that
46 /// have the same physical dimension set of T^(-1)Â·L, which is the physical dimension set of speed,
47 /// so these two units of measure can be converted between one another. On the other hand, the
48 /// kilogram per cubic metre is a unit of measure with physical dimension set L^(-3)Â·M, which is the
49 /// physical dimension set of mass density, so this unit of measure cannot be converted to either
50 /// the metre per second or the mile per hour, which have a different physical dimension set.
